04 · 3D Character · Fan Art

Roller Jousting — Eleanor Fan Art

This 3D character is my personal interpretation based on an original concept by Johnson Ting. I loved the design language and atmosphere of the concept, and set out to translate it into a fully realized 3D version while adding my own artistic touch.

The sculpt was created in ZBrush, focusing on strong forms, clean silhouettes, and detailed hard surface work. Retopology was handled in Maya to ensure a clean, animation-friendly mesh suitable for real-time use. All textures were created in Substance Painter using a PBR workflow.

Thanks for taking the time to check out this project. I hope you enjoy it as much as I enjoyed bringing it to life!

05 · 3D Character · AAA Realtime

RealTime Northern Viking

This personal project features a warrior character inspired by Norse aesthetics, designed for integration into AAA or indie video game environments. The hair has been created using hair cards to ensure real-time performance, and the entire retopology has been carefully crafted with optimization in mind.

Despite being an AAA-quality character, the model is fully prepared for efficient implementation in modern game engines, maintaining a balance between visual fidelity and technical performance. Hope you like it, and any feedback is welcome.
